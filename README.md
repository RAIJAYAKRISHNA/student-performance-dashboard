# Student Performance Analytics Dashboard

## Project Overview

This project analyzes student academic performance using SQL Server and Microsoft Excel. The objective is to evaluate student scores, identify top performers, analyze subject-wise performance, and visualize insights through an interactive dashboard.

---

## Tools Used

* SQL Server
* Microsoft Excel

---

## Database Structure

### Students

Contains student information such as Student ID, Student Name, and Gender.

### Subjects

Contains subject details.

### Marks

Stores marks obtained by students in different subjects.

---

## SQL Analysis Performed

* Total Students
* Subject-wise Average Marks
* Top 5 Students by Total Score
* Student Ranking using RANK() Window Function
* Gender Distribution
* Pass Percentage Analysis
* Average Score Calculation

---

## Dashboard KPIs

* Total Students: 8
* Highest Score: 378
* Average Marks: 82.81
* Pass Percentage: 100%

---

## Dashboard Features

* KPI Cards
* Subject-wise Average Marks Analysis
* Top 5 Students Analysis
* Gender Distribution Analysis
* Interactive Subject Slicer
* Interactive Student Slicer
* Pivot Tables and Pivot Charts

## Key Insights

* Anjali achieved the highest total score of 378.
* Python recorded the highest average marks among all subjects.
* Male and Female student counts are evenly distributed.
* Overall pass percentage is 100%.
* The top-performing students consistently scored above the overall average.
